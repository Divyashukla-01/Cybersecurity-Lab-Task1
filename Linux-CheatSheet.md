# Linux Basic Commands Cheat Sheet

## File & Directory Commands
pwd – Show current directory  
ls – List files  
mkdir – Create directory  
cd – Change directory  
touch – Create file  

## Permission Commands
chmod – Change file permissions  

## Network Commands
ifconfig – Show IP address  
ping – Test connectivity  

## Security Tools Commands
nmap – Scan open ports  
openssl – Generate hash (MD5, SHA256)  
nc – Establish network communication  

---

This cheat sheet covers basic Linux commands used in the cybersecurity lab.
